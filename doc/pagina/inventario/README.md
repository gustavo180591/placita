# Inventario

Esta carpeta contiene la documentación para la gestión de inventario, incluyendo productos, stock y alertas.

## Funcionalidades Principales

- Gestión de productos y categorías.
- Control de stock y niveles mínimos.
- Alertas de inventario bajo.
- Sincronización con compras y ventas.
