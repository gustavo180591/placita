# Clientes

Esta carpeta contiene la documentación para la gestión de clientes, incluyendo registro, perfiles, historial de compras y políticas de fiado.

## Funcionalidades Principales

- Registro y edición de perfiles de clientes.
- Historial de transacciones y compras.
- Gestión de créditos y políticas de fiado.
- Búsqueda y filtrado de clientes.
