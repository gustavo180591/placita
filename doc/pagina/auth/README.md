# Auth - Autenticación

Esta carpeta contiene la documentación para las páginas de autenticación, incluyendo login, registro, recuperación de contraseña y gestión de sesiones de usuario.

## Funcionalidades Principales

- Inicio de sesión con credenciales.
- Registro de nuevos usuarios.
- Recuperación y cambio de contraseñas.
- Gestión de sesiones y tokens JWT.
