Dirección y Gobierno

Director/a de Programa (Program Director) — PD
Qué hace: Alinea negocio-producto-tech, define alcance y prioriza valor.
Tareas: roadmap integral, gestión de riesgos, gobernanza, OKRs, presupuesto.

Gerente/a de Proyecto (Project Manager) — PM
Qué hace: Plan, cronograma y ejecución.
Tareas: Gantt/sprints, dependencias, seguimiento, reportes, desbloqueos.

Propietario/a de Producto (Product Owner) — PO
Qué hace: Dueño/a de backlog y valor al cliente.
Tareas: historias, priorización, aceptación, KPIs de producto.

Scrum Master / Agile Coach — SM
Qué hace: Salud del proceso ágil.
Tareas: ceremonias, métricas (lead time, flow), mejora continua, cultura.

Producto, UX y Contenido

Diseñador/a UX Research — UXR
Qué hace: Investiga necesidades reales.
Tareas: entrevistas, tests de usabilidad, journey maps, insights accionables.

Diseñador/a de Producto (UI/UX) — Product Designer
Qué hace: Diseña experiencias y pantallas.
Tareas: wireframes, prototipos, design system, accesibilidad.

Redactor/a UX (UX Writer) — UXW
Qué hace: Microcopys que convierten.
Tareas: textos de flujos, vacíos de estado, tono de marca.

Gestor/a de Catálogo & Contenido — Catalog Manager
Qué hace: Datos de productos impecables.
Tareas: atributos, fotos, descripciones, variantes, SEO on-page.

Especialista SEO Técnico — SEO
Qué hace: Indexación y performance orgánica.
Tareas: schema, sitemaps, core web vitals, enlaces internos.

Ingeniería (Arquitectura & Plataforma)

Arquitecto/a de Soluciones — SA (Solutions Architect)
Qué hace: Plano maestro técnico.
Tareas: decisiones tech, diagramas, no-funcionales (escala, latencia, HA).

Líder Técnico/a (Tech Lead) — TL
Qué hace: Calidad técnica del equipo.
Tareas: revisiones, estándares, mentoring, decisiones de diseño.

Desarrollador/a Frontend Web — FE
Qué hace: Tienda web (SPA/SSR).
Tareas: UI, estado, accesibilidad, performance.

Desarrollador/a Mobile (iOS/Android/Flutter) — Mobile
Qué hace: App nativa/híbrida.
Tareas: vistas, notificaciones, deep links, stores.

Desarrollador/a Backend (API/Servicios) — BE
Qué hace: Lógica de negocio y APIs.
Tareas: dominios (carrito, checkout), integraciones, seguridad, caching.

Ingeniero/a de Datos — DE
Qué hace: Data pipes confiables.
Tareas: ETLs/ELTs, modelado (DW/Lakehouse), calidad y linaje.

Científico/a de Datos / ML Engineer — DS/ML
Qué hace: Modelos que venden.
Tareas: recomendadores, búsqueda semántica, pricing dinámico, forecasts.

Ingeniero/a DevOps / SRE — DevOps/SRE
Qué hace: Disponibilidad y despliegues.
Tareas: CI/CD, IaC, observabilidad, autoscaling, DR.

Administrador/a de Base de Datos — DBA
Qué hace: Datos sanos y rápidos.
Tareas: índices, particionamiento, tuning, backups/restore.

Ingeniero/a de Integraciones — Integrations
Qué hace: Conecta todo con todo.
Tareas: PSPs, ERP, WMS, 3PL, marketplaces, facturación, webhooks.

Calidad, Seguridad y Cumplimiento

QA Automation Engineer — QA Auto
Qué hace: Calidad continua.
Tareas: test e2e/API/UI, regresión, pipelines de calidad, cobertura.

QA Manual / Exploratorio — QA Manual
Qué hace: Descubre lo que los scripts no ven.
Tareas: casos límite, compatibilidad, UAT, smoke tests.

Release Manager — Release
Qué hace: Orquesta versiones sin drama.
Tareas: ramas, freezes, changelogs, rollbacks.

Especialista en Seguridad Aplicativa — AppSec
Qué hace: Seguridad desde el diseño.
Tareas: threat modeling, SAST/DAST, hardening, secretos.

Especialista Compliance (PCI/Privacy) — Compliance
Qué hace: Cumplir sin frenar.
Tareas: PCI DSS, políticas de datos, DPIA, cookies/consent.

Ingeniero/a Anti-Fraude / Riesgo — Fraud/Risk
Qué hace: Protege margen y clientes.
Tareas: reglas, scoring, chargebacks, señales de riesgo.

Pagos y Monetización

Líder de Pagos — Payments Lead
Qué hace: Cobrar siempre, cobrar bien.

Analista de Cobros & Chargebacks — Collections
Qué hace: Minimiza pérdidas.
Tareas: disputas, evidencia, tasas de aprobación.

Analista de BI / Analytics — BI

Especialista en Performance Ads — Paid Media
