# Glosario

- **Fiado**: venta a crédito informal, saldo pendiente a un cliente.
- **Arqueo**: conciliación de efectivo esperado vs contado.
- **Merma**: baja de stock por vencimiento/defecto.
- **Reserva/Seña**: compromiso de stock con pago parcial.
- **Combo**: SKU virtual que descuenta múltiples productos.
- **QR pendiente**: pago electrónico registrado aún no confirmado.
