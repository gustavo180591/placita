# Roadmap

## MVP (4–6 semanas estimativas de desarrollo efectivo)
- POS básico (ventas, efectivo/QR/fiado), inventario, clientes, compras, reportes esenciales, offline, exportación.
- Conciliación QR manual; tickets PDF simples.

## V2
- Promos por cantidad, combos, multiusuario básico, impresión Bluetooth, notificaciones.

## V3
- Precios por medio de pago, actualización masiva, dashboard web, WhatsApp, sugerencias de reposición.
