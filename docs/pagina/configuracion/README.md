# Configuración

Esta carpeta contiene la documentación para las páginas de configuración de la aplicación, incluyendo ajustes generales y parámetros.

## Funcionalidades Principales

- Configuración de parámetros de la aplicación.
- Ajustes de tienda y sucursales.
- Configuración de impuestos y políticas.
- Gestión de usuarios y permisos.
