# Ventas

Esta carpeta contiene la documentación para el módulo de ventas y punto de venta (POS), incluyendo procesos de venta, tickets y sincronización.

## Funcionalidades Principales

- Interfaz de punto de venta (POS).
- Procesamiento de ventas y generación de tickets.
- Gestión de pagos y métodos.
- Sincronización offline y reportes de ventas.
