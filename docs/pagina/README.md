# Documentación de Páginas

Esta carpeta contiene la documentación específica para cada módulo o página de la aplicación Placita. Cada subcarpeta corresponde a una sección funcional de la aplicación, como autenticación, ventas, inventario, etc.

## Estructura

- **auth/**: Páginas relacionadas con autenticación y gestión de usuarios.
- **clientes/**: Funcionalidades para gestionar clientes.
- **ventas/**: Procesos de ventas y punto de venta (POS).
- **inventario/**: Gestión de productos e inventario.
- **compras/**: Procesos de compra a proveedores.
- **productos/**: Catálogo de productos.
- **proveedores/**: Gestión de proveedores.
- **reservas/**: Sistema de reservas.
- **fiado/**: Gestión de créditos o fiados.
- **configuracion/**: Ajustes y configuración de la aplicación.
- **ajustes/**: Configuraciones adicionales.
- **devoluciones/**: Procesos de devoluciones.
- **estado/**: Estados de la aplicación o módulos.
- **exportar/**: Funcionalidades de exportación de datos.
- **importar/**: Funcionalidades de importación de datos.
- **notificaciones/**: Sistema de notificaciones.
- **offline/**: Funcionalidades para modo offline.
- **reportes/**: Generación de reportes.

Para más detalles sobre cada módulo, consulta los archivos en las subcarpetas correspondientes y la documentación general en `/doc/`.
