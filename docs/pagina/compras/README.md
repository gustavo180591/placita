# Compras

Esta carpeta contiene la documentación para el módulo de compras, incluyendo órdenes, recepción y proveedores.

## Funcionalidades Principales

- Creación y gestión de órdenes de compra.
- Recepción de mercancía.
- Integración con inventario y proveedores.
- Reportes de compras.
